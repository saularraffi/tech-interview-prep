# System Design

## Flshcard Approach (using Anki)

Instead of memorizing facts, practice decision making. 4 Types of cards to facilitate this:

1. Tradeoff cards - when to use X over Y...pros and cons using Z
2. "When would you use X?" cards - what signals indicate X needs to be used
3. Failure mode cards - what can go wrong when using X
4. System walkthrough cards - what are the main components required to build X
